Report on Semantic Table and Form Tags

This report discusses the semantic tags used in the table and form sections of the code.

Table Tags

  • <table> - Defines a structured data table.
  • <caption> - Provides a title for the table.
  • <thead> - Groups the header row content.
  • <tbody> - Contains the main table data.
  • <tr> - Represents a row in the table.
  • <th> - Defines header cells (bold & centered).
  • <td> - Represents standard data cells.

Form Tags

  • <form> - Defines an input submission form.
  • <fieldset> - Groups related input fields.
  • <legend> - Provides a title for the fieldset.
  • <label> - Defines input field descriptions.
  • <input> - Accepts various user inputs.
  • <select> - Creates a dropdown menu.
  • <option> - Represents choices in a dropdown.
  • <button> - Submits form data.
